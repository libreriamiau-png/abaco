🔄 Migrar Códigos de Compra

📋 ¿Qué hace este script?
Extrae los códigos de compra/lote del campo localiz y los separa en campos dedicados.

Ejemplo: "9g 26HIG" → localiz: "9g", compra_codigo: "26HIG"
Patrón detectado: espacio + 2 dígitos + letras mayúsculas (ej: 25HIG, 26TEO, 24DIC)
⚠️ IMPORTANTE:
• Este script modifica la base de datos
• Revisa la vista previa antes de aplicar cambios
• Se recomienda hacer backup antes (aunque no se borran datos, solo se reorganizan)
